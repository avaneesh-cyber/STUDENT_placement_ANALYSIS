# 🎓 Student Placement Prediction & Skill Gap Analysis

## 📌 Project Overview
This project predicts student placement using Machine Learning and analyzes skill gaps.

## 🚀 Features
- Placement Prediction Model (Random Forest)
- Skill Gap Analysis
- Data Visualization (EDA)
- Streamlit Web App
- Power BI Dashboard

## 🛠 Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Streamlit
- Power BI

## 📊 Results
- High model accuracy
- Communication Skills & CGPA most important
